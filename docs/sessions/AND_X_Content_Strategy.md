## 1. 계정 정체성

**계정명**: AND센터 | DailyNK (@ANDCenter_NK)  
**계정 미션**: 북한 실상을 데이터로 기록하고, 인권·노동·통제 구조를 분석해 국제사회에 전달

**포지셔닝**:
- ❌ 선정적 뉴스 계정
- ❌ 정치 선동 계정
- ✅ **데이터 저널리즘 연구기관** (BBC Monitoring, NK News 스타일)

**차별화 포인트**:
- 38North: 군사·핵 중심 → AND센터: 민생·경제·사회 데이터
- NK News: 종합 뉴스 → AND센터: 분석 위주
- DailyNK 본계정: 속보 중심 → AND센터: 구조적 해석

## 2. 타겟 독자층 (3-Tier)

### Tier 1: 해외체류 조선 주민
- **언어**: 한국어 우선
- **관심사**: 고향 소식, 민생 변화, 실질적 생활 정보
- **톤**: 사실 전달, 과도한 감정 배제

### Tier 2: 북한 연구자·정책 전문가
- **언어**: 영어 우선 (또는 한영 병기)
- **관심사**: 데이터 신뢰성, 출처, 분석 방법론
- **톤**: 학술적, 중립적 관찰자

### Tier 3: 일반 국제 독자
- **언어**: 영어
- **관심사**: 북한 실상에 대한 호기심, 인권 이슈
- **톤**: 명확하고 간결한 사실 전달

## 3. 작성 원칙 (Core Principles)

### 3.1 데이터 우선주의
**✅ 좋은 예시**:
- "평양 쌀값 kg당 6,200원, 전월 대비 15% 상승 (DailyNK 시장 조사)"
- "위성 이미지 분석 결과, 함경북도 00군 건설 현장 3개월간 활동 중단 확인"

**❌ 나쁜 예시**:
- "김정은이 숨기려 했던 충격적 사실"
- "북한 주민들의 분노가 폭발 직전"

### 3.2 출처 명시 원칙
**필수 포함 요소**:
- DailyNK 내부 소식통
- 위성 분석 (출처: Planet Labs, Sentinel 등)
- 탈북민 증언
- 공식 문건 (조선중앙통신, 노동신문 등)

**표현 방식**:
- "DailyNK 취재 결과" ✅
- "내부 폭로" ❌
- "최근 확인된 문건" ✅
- "최초 공개" ❌ (과장 금지)

### 3.3 중립적 관찰자 톤
**비교 프레임**:
| 선정적 표현 (❌) | 분석적 표현 (✅) |
|----------------|----------------|
| "체제의 잔혹성" | "국제 규범과 상반되는 정책" |
| "주민들의 고통" | "민생 지표 악화" |
| "독재 정권" | "중앙집권 체제" |

## 4. 포스트 구조 (280자 제한)

### 4.1 기본 템플릿

```
[핵심 사실 1문장 - 데이터 포함]

[공백 1줄]

[맥락/의미 해석 1~2문장]

[공백 1줄]

[원문 링크]

[공백 1줄]

[해시태그 2~3개 - 반드시 마지막 줄]
```

**포맷 규칙**:
- 문장 사이 **공백 1줄 필수** (X 알고리즘 가독성 최적화)
- 해시태그는 **무조건 마지막 줄** (본문 가독성 보호)
- 수치 상승/하락 시 이모지 사용 (선택):
  - 상승: ▲ 또는 📈
  - 하락: ▼ 또는 📉
  - 예: "쌀값 ▲15%"

### 4.2 실전 예시

**한국어 포스트** (Tier 1 대상):
```
평양 쌀값 kg당 6,200원, 전월 대비 15% 상승.

DailyNK 시장 조사 결과, 공급망 차단으로 가격 급등. 주민 구매력 약화 우려.

https://www.dailynk.com/...

#북한 #경제 #DailyNK
```

**영어 포스트** (Tier 2-3 대상):
```
Rice prices in Pyongyang surge 15% to ₩6,200/kg.

DailyNK market survey indicates supply chain disruption. Concerns over declining purchasing power.

https://www.dailynk.com/...

#NorthKorea #Economy #HumanRights
```

**이중 언어 포스트** (전 독자 대상):
```
평양 쌀값 15% 급등 (Rice prices up 15%)

공급망 차단으로 kg당 6,200원 기록. 주민 구매력 약화 우려.
DailyNK survey shows supply disruption impact.

https://www.dailynk.com/...

#북한 #NorthKorea #DailyNK
```

**🔴 중요**: 이중 언어 포스트는 280자 초과 위험이 높음.
- 영문 요약은 **100자 이내로 제한**
- 또는 **주요 수치만 병기** (예: "15% 급등 (up 15%)")
- 링크는 단축 URL 사용 시 23자로 카운트됨

## 5. 텍스트 기반 데이터 시각화

이미지 첨부 없이 **텍스트만으로 데이터 카드 효과** 구현:

### 5.1 불렛 포인트 스타일 (레이아웃 A)

```
📊 DailyNK 시장 분석

• 품목: 쌀
• 가격: 6,200원/kg
• 변동: +15% (전월 대비)
• 원인: 국경 봉쇄 장기화

https://www.dailynk.com/...
```

### 5.2 비교 테이블 스타일 (레이아웃 B)

```
평양 vs 지방 쌀값 격차 심화

평양: 6,200원
신의주: 8,500원
차이: 37%

지역별 공급 불균형 가속화.

https://www.dailynk.com/...
```

### 5.3 단순 서술형 (레이아웃 C)

```
DailyNK 시장 조사 결과, 평양 쌀값이 kg당 6,200원으로 전월 대비 15% 상승했다.

국경 봉쇄 장기화로 공급망 차단이 주요 원인.

https://www.dailynk.com/...
```

**🔴 스팸 방지 필수 조치**:
- 동일 레이아웃 연속 사용 금지
- 3가지 레이아웃 중 **무작위 선택** 또는 **순환 사용**
- "📊 DailyNK 분석" 같은 시작 문구 매번 다르게 변형

## 6. 금지 사항 (Prohibited Practices)

### 6.1 절대 금지
- ❌ "공유해주세요", "리트윗 부탁드립니다" (자동화 스팸 필터)
- ❌ 확인 불가능한 추측 ("~일 것으로 보인다", "~한 것으로 추정")
- ❌ 과도한 이모지 (📢❗🔥 등)
- ❌ 선정적 단어 ("충격", "경악", "폭로", "대참사")

### 6.2 지양 사항
- 🟡 감정 자극 표현 ("분노", "공포", "절망")
- 🟡 장문 스레드 (3개 이상)
- 🟡 해시태그 과다 (4개 이상)

## 7. 권장 사항 (Best Practices)

### 7.1 숫자 우선
- "3배 증가" > "크게 늘었다"
- "70% 감소" > "대폭 줄었다"
- "2,000명 동원" > "많은 주민 동원"

### 7.2 비교 프레임
- "전년 대비", "전월 대비"
- "평양 vs 지방"
- "공식 발표 vs 실제 조사"

### 7.3 질문형 마무리 (선택)
**적절한 사용**:
- "이는 무엇을 의미하나?" ✅
- "당신의 생각은?" ❌ (직접 CTA)

## 13. 포스팅 스케줄 전략 (Time-Content Matching)

### 13.1 현재 Cron 설정 (KST 기준)

| 회차 | KST | 미 동부 | 유럽(GMT) | 타겟 독자 |
|------|-----|---------|-----------|----------|
| 1차 | 08:10 | 전날 18:10 | 전날 23:10 | 🇰🇷 국내 연구자 (출근길) |
| 2차 | 10:25 | 전날 20:25 | 00:25 | 🇰🇷 국내 전문가 (업무 중) |
| 3차 | 12:40 | 전날 22:40 | 02:40 | 🇰🇷 일반 독자 (점심시간) |
| 4차 | 15:00 | 01:00 | 06:00 | 🇪🇺 유럽 (새벽) |
| 5차 | 17:10 | 03:10 | 08:10 | 🇪🇺 유럽 (아침) |

**문제점**:
- ❌ 미국 정책 관계자 완전 누락 (전부 밤/새벽)
- ❌ 해외체류 조선 주민 시간대 미고려
- ✅ 유럽은 5차(17:10)만 적합

### 13.2 타겟별 골든 타임

**Tier 1: 해외체류 조선 주민**
- 주요 거주지: 중국(동북3성), 동남아, 러시아 극동
- 골든 타임: 현지 저녁 20:00~23:00
- 중국 기준: KST 21:00~24:00 (1시간 시차)
- 동남아 기준: KST 22:00~01:00 (2시간 시차)

**Tier 2: 미국 정책 관계자**
- 주요 타겟: 워싱턴 DC, 뉴욕
- 골든 타임: 미 동부 09:00~11:00 (업무 시작)
- KST 변환: **22:00~24:00** (다음날 00:00~02:00)

**Tier 3: 유럽 연구자**
- 주요 타겟: 런던, 브뤼셀, 베를린
- 골든 타임: GMT 08:00~10:00
- KST 변환: **17:00~19:00**

### 13.3 시간대별 컨텐츠 전략

**골든 타임 1 (KST 08:10)**
- **타겟**: 🇰🇷 국내 연구자 + 🇪🇺 유럽 자정 독자
- **컨텐츠**: 데이터 집약형 (레이아웃 A: 불렛 포인트)
- **언어**: 한국어 우선 + 영문 요약

**골든 타임 2 (KST 17:10)**
- **타겟**: 🇰🇷 퇴근길 + 🇪🇺 유럽 오전
- **컨텐츠**: 이중 언어 포스트 (글로벌 도달 극대화)
- **언어**: 한영 병기

**🔴 누락된 골든 타임 (향후 추가 권장)**
- **KST 22:00~24:00** (미 동부 09:00~11:00)
- **타겟**: 🇺🇸 워싱턴 정책 관계자 + 해외체류 주민
- **컨텐츠**: 영어 우선 포스트 + 인사이트 도출형 스타일
- **전략**: 주요 분석 기사 배정

### 13.4 X 알고리즘 고려사항

**포스팅 간격**:
- ✅ 최소 1시간 이상 간격 유지 (현재 Cron 적합)
- ❌ 1시간 내 2개 이상 → 도배(Flood) 감지 위험

**시간대별 우선순위**:
1. **Priority 1 (08:00~10:00 KST)**: 고품질 데이터 분석
2. **Priority 2 (17:00~19:00 KST)**: 글로벌 독자 타겟 (이중 언어)
3. **Priority 3 (22:00~24:00 KST)**: 미국 타겟 (영어 중심) ← 현재 누락

### 13.5 개선 제안 (Phase 2 확장)

**단기 (현재 Cron 유지)**:
- 5차 포스팅(17:10)을 **이중 언어 + 글로벌 타겟**으로 특화
- 1차 포스팅(08:10)을 **국내 전문가 타겟** 데이터 중심으로 특화

**중기 (Cron 1회 추가)**:
- **KST 22:30** 추가 (미 동부 09:30)
- 하루 6건으로 확대 (X API Free tier 500트윗 여유 있음)
- 영어 포스트 전용 시간대

**장기 (동적 스케줄링)**:
- 기사 중요도에 따라 골든 타임 배정
- A/B 테스트 결과 기반 시간대 최적화
- 주말 스케줄 별도 운영 (현재는 평일만)

### 13.6 Claude API 프롬프트 통합

**시간대별 프롬프트 변형** (선택):

```python
# api/cron.py 예시

TIMESLOT_STRATEGIES = {
    "08:10": {
        "target": "국내 연구자",
        "style": "학술적 요약형",
        "language": "한국어 우선",
        "layout": "A"  # 불렛 포인트
    },
    "17:10": {
        "target": "글로벌 독자",
        "style": "인사이트 도출형",
        "language": "이중 언어",
        "layout": "B"  # 비교 테이블
    },
    "22:30": {  # 향후 추가
        "target": "미국 정책 관계자",
        "style": "데이터 중심",
        "language": "영어",
        "layout": "A"
    }
}

def get_timeslot_strategy(current_hour: int) -> dict:
    """현재 시간에 맞는 컨텐츠 전략 반환"""
    for timeslot, strategy in TIMESLOT_STRATEGIES.items():
        hour = int(timeslot.split(":")[0])
        if abs(current_hour - hour) < 1:
            return strategy
    
    # 기본 전략
    return TIMESLOT_STRATEGIES["08:10"]
```
